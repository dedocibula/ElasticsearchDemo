Welcome. We prepared this short demo to help you master the basics of Elasticsearch. Throughout the course of this lecture you will have chance to play with real ES cluster. This tutorial will walk you through some fundamental as well as more advanced operations.

To interact with Elasticsearch you will be using lightweight user-friendly Javascript console called Sense. Sense allows contains many features allowing you to benefit from real-time command execution, result visualization and autocompletion.

There are 3 practice examples. Feel free to do them in any order you like, however note that some advanced examples might build upon topics that were already covered in previous lessons.
